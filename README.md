# Anuvansh.OS Portfolio

An interactive, OS-themed portfolio website built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- **Draggable Windows**: Interact with project details using physics-based dragging.
- **Micro-Interactions**: Spring animations, magnetic buttons, and hover effects.
- **Gravity Effect**: Shake windows to make content fall!
- **Scanning Laser**: Special visualization for Easy PDF project.
- **Morse Code Reveal**: Interactive text reveal for hardware projects.
- **Space-Themed**: Blueprint aesthetic with technical details.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **State Management**: Zustand
- **Icons**: Lucide React
- **Fonts**: Inter, JetBrains Mono

## Project Structure

- `/app` - Next.js App Router pages and layout
- `/components` - React components (Windows, Desktop, etc.)
- `/stores` - Zustand state stores
- `/utils` - Animation variants and physics calculations

## Development

- `npm run build` - Build for production
- `npm run lint` - Run ESLint
