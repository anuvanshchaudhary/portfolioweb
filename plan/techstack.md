1. The Core Framework: Next.js (App Router)
Why: Since you’ve already used Next.js for "Easy PDF," continuing with it for your portfolio demonstrates deep expertise rather than just surface-level knowledge.

Key Benefit: It provides excellent SEO out-of-the-box through Server-Side Rendering (SSR), ensuring your portfolio (and name) is discoverable by HRs on search engines.

Suitability: It handles everything from simple static pages to complex full-stack features if you decide to add a blog or interactive elements later.

2. Styling & UI: Tailwind CSS + shadcn/ui
Tailwind CSS: A "utility-first" framework that allows you to build custom, responsive designs very quickly. It’s a standard in modern tech companies.

shadcn/ui: Instead of a traditional component library, this is a collection of reusable components that you "own" (copy-paste into your project). It’s built on top of Tailwind and Radix UI, offering a high-end, professional look that is very popular in 2024–2025.

3. Animations: Framer Motion
Why: For a React/Next.js site, Framer Motion is the industry standard for adding "polish." It’s declarative, making it easy to create smooth transitions when users scroll through your projects or hover over cards.

HR Impact: Subtle animations (like a project card gently lifting on hover) make the site feel high-quality and "expensive," which leaves a lasting impression on recruiters.

4. Backend & Database (If needed): Supabase or Prisma + MongoDB
Supabase: If you want to add a guestbook, a live "tech stack" tracker, or a contact form that saves to a database, Supabase (PostgreSQL) is the easiest "Backend-as-a-Service" to integrate with Next.js.

Alternative: Since your resume mentions MySQL and PostgreSQL, using Prisma (an ORM) with a database of your choice is a great way to showcase your backend/database management skills.

5. Deployment & Tools
Deployment: Vercel is the natural home for Next.js. It offers fast global delivery and automatic "Preview Deployments" for every GitHub commit you make.

Analytics: Vercel Analytics or PostHog (free tiers) will let you see if HRs from specific companies are visiting your site and which projects they are clicking on.
